Font can be changed incrementally or to specific sizes.

Items marked in red are DEFAULTS which can be adjusted easily.

  • For incremental changes:
    • Any element with class name increaseClickItem will increase the font size.
    • Any element with class name decreaseClickItem will decrease the font size.
    • The Default increment value is 2. See below for example of ways to change the options.

When the minFont or maxFont values are reached, class DISABLED will be passed to increaseClickItem / decreaseClickItem classes, respectively. The same is true if the fontResizer cookie returns a value that matches minFont or maxFont.

  • For specific changes:
    • Any element with class name setFontSize will set the font size.
    • The size to be set is determined via the title attribute which needs to be passed when the class is set.
    • Example: <a class="setFontSize" title="20"> will set font size to 20.

When an element with class setFontSize is clicked, class ON will be passed to it. When another element with class setFontSize is clicked, ON will be removed from the previous element. This also occurs when increaseClickItem / decreaseClickItem elements are clicked and reach a value that matches a setFontSize elements value. The same is true if the fontResizer cookie returns a value that matches a setFontSize value.

General Defaults are as follows:

  • sizeType: 'px'
  • minFont: 12
  • maxFont: 16
  • fontSet: 14
  • setFontOn: 'ON'
  • increment: 2
  • incrementDisable: 'DISABLED'

Cookie Defaults are as follows:

  • cookieName: 'fontResizer'
  • cookieExpire: cExpireDate
    • cExpireDate is set to the Month/Day the cookie is set, one year from it's creation.

    Variables used for cExpireDate: (override as needed for alternate cookie expiration or create new)
    var date = new Date();
    var currYear = date.getFullYear();
    var currMonth = date.getMonth();
    var currDay = date.getDate();
    var cExpire = new Date(currYear + 1, currMonth, currDay);
    var cExpireDate = cExpire.toUTCString();

  • cookiePath: '/'

Implementation consists of calling the fontResizer function on a comma-seperated list of elements:
Example:
$(document).ready(function() {
    $('#content p, .resizeMe, .resizeSpan').fontResizer({minFont:10,maxFont:20,increment:1});
});
Note that Default Overrides (OPTIONS) are passed as params within curly braces. Pay close attention to quote inclusion. When in doubt refer to the Default lists above.
